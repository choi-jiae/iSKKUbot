# iSKKUbot
**iSKKUbot**은 **교환 학생을 위한 학교 정보 제공 챗봇**으로 캡스톤 프로젝트 수업에서 진행한 프로젝트입니다.
- **RAG**를 사용하여 좀 더 자유롭게 질문할 수 있는 동시에 정확한 정보를 얻을 수 있는 챗봇을 구현했습니다.
- web 데이터의 경우 매일 특정 시간에 웹 스크래핑을 시도하여 항상 **최신의 정보**를 얻을 수 있도록 하였습니다.
